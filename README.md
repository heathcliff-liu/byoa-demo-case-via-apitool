# byoa_demo_case_via_apitool

A minimal **BYOA (Bring Your Own Agent)** demo deployed on SAP BTP Cloud Foundry.

The agent exposes a 3PL billing validation assistant via the [A2A protocol](https://github.com/google/a2a), powered by **Zhipu GLM-4-Flash** and hardcoded mock data вҖ” no SAP AI Core, no XSUAA, no OData required.

---

## Architecture

```
HTTP Client (curl / Bruno / Joule)
        в”Ӯ  POST /  (A2A JSON-RPC)
        в–ј
__main__.py  в”Җв”Җ A2AStarletteApplication (uvicorn)
        в”Ӯ
        в–ј
executor.py  в”Җв”Җ InvoiceAgentExecutor
        в”Ӯ
        в–ј
agent.py     в”Җв”Җ LangGraph two-node graph
        в”ңв”Җв”Җ node "agent"   : System prompt + mock data вҶ’ GLM-4-Flash
        в””в”Җв”Җ node "respond" : Classify status (completed / input_required / error)
```

**Key dependencies:**
- `a2a-sdk` вҖ” A2A HTTP server, compatible with Joule
- `langgraph` + `langchain-openai` вҖ” Agent graph, calls Zhipu via OpenAI-compatible API
- `uvicorn` вҖ” ASGI server

## Local Development

```bash
git clone <this-repo>
cd byoa_demo_case_via_apitool

# create venv and install deps
uv venv && source .venv/bin/activate
uv sync

# set env vars
export ZHIPU_API_KEY=<your-key>
export USE_MOCK_DATA=true

# run
python -m src
# Agent starts at http://localhost:8080
```

Verify:
```bash
curl http://localhost:8080/.well-known/agent.json
```

## Deploy to SAP BTP Cloud Foundry

### 1. Login

```bash
cf login -a https://api.cf.us10-001.hana.ondemand.com --sso
```

### 2. Set your API key in manifest.yml

Edit `manifest.yml` and replace `YOUR_ZHIPU_API_KEY` with your real key:

```yaml
env:
  ZHIPU_API_KEY: <your-key>
```

> Do **not** commit real keys to git. Use `cf set-env` instead if preferred:
> ```bash
> cf set-env byoa-coach-demo ZHIPU_API_KEY <your-key>
> ```

### 3. Export requirements (if pyproject.toml changed)

```bash
uv export --format requirements-txt --no-hashes -o requirements.txt
```

### 4. Push

```bash
cf push
```

## Troubleshooting

| Problem | Fix |
|---------|-----|
| `GLM API error 401` | Check `ZHIPU_API_KEY` is set correctly |
| `ModuleNotFoundError` | Re-run `uv export` to regenerate requirements.txt |
| `Env cannot set PORT` | Remove `PORT` from manifest.yml вҖ” CF injects it automatically |
| App crashes on CF | Increase memory to `512M` in manifest.yml |
| Bruno `Extra data` JSON error | Check the request body ends with a single `}` |
